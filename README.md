# Hulbert-General-Store

헐버트 잡화점!

**경복궁 옆 서촌 한옥**을 무대로, 1인 핸드메이드 작가의 **'제품'이 아니라 '사람과 꿈'에 투자**하는 상생형 크래프트 리테일 공동체입니다. 위탁(복지수수료) → 직원(너투사) → 주주(나투사)로 작가와 함께 성장합니다.

이름 '헐버트'는 한글을 사랑하고 한국 독립을 도운 미국인 **호머 헐버트 박사**(1863~1949)에서 따왔습니다 — 앞에 나서지 않고 작가를 뒤에서 받친다는 우리의 정신을 담았습니다.

## 문서

> 미정 수치는 `예시(제안)`, 민감·법적 항목(파손 책임·보험·지분·법인 등)은 `협의/TBD`로 표기. 계약·법인·상표 관련은 실제 진행 전 전문가(변호사·변리사·세무사) 검토 권장.
