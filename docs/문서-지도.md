## 0. 두 이름 먼저 — 헷갈리지 않게

| 이름 | 무엇 | 누구를 위해 |
| --- | --- | --- |
| **헐버트잡화점** | 경복궁 옆 서촌 한옥 **매장(사업체)** — 작품이 팔리는 곳, '입점'의 대상 | 손님 + (사업 운영) |

## 3. ✅ 확정 핵심 사실 (단일 출처 — 모든 문서가 이걸 따름)

> 새 문서를 쓰거나 고칠 때 이 표와 어긋나면 안 됩니다.

| 항목 | 확정값 |
| --- | --- |
| 입점비 | **0원** |
| 가격 결정 | **100% 작가** |
| 수수료 | 증명 전 **일반 30%** → 일정 매출(게이트=내부 재량) 후 **누진 복지수수료(30%↓)** |
| 수수료 성격 | 회사 마진 아님 → **한옥 고정비로 전액** |
| 한옥 | **100평대**, 월세 **500만 원 이상** |
| 법인 | **주식회사**(+주주간계약) *잠정* |
| 작품 책임 | 통상관리·경미 손상 **회사 부담**, 고가품 **별도 합의**, 불가항력 면책 |
| 3단계 | 위탁 → 너투사(3.3%) → 나투사(주주). **사다리는 선택** |
| 호칭 | 나이·학벌 안 물음 · **꿈을 담은 한글 이름** |
| 취급 | **장르 불문** — '한옥에 어울리면' |
| 가치/톤 | 핵심가치 **함께 · 공평 · 투명** (+ 정직한 '목소리') · '모시다' 금지 |
| 제1약속 | *"잃을 걱정 없이, 하고 싶은 창작만 — 잘 팔리면 복지로 보답, 끝내 함께 주인."* |

> 운영 수치(정산 월 1회 · 로테이션 4주 · 퇴점 8주+사전안내 등)는 **예시(제안)** — 운영하며 조정.
