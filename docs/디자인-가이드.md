# 헐버트작가촌 사이트 — 디자인 가이드

> 🏷️ **📝 초안** · 갱신 2026-07 · 디자인 정본 (v0.1)

> [`브랜드-정의서`](브랜드-정의서.md) §8의 비주얼 **무드**("한지 아이보리 · 먹빛 · 절제된 단청 적 · 청자 톤 · 자연광 · 여백 · 손맛")를 실제로 구현 가능한 **토큰·타이포·컴포넌트 규격**으로 확정합니다.
> [`문서-지도`](문서-지도.md) §5가 1순위로 꼽았던 'UI/UX 디자인 가이드'가 이 문서입니다.

**버전 v0.1** &nbsp;|&nbsp; ⚠️ 색상 값은 **예시(제안)** — 문서 뷰어에서 이미 검증된 팔레트를 기준으로 삼았고, 실물(한지·한옥 사진)과 대조해 확정합니다.

---

## 1. 컬러 토큰 *(값은 예시·제안)*

| 토큰 | 값 | 무드 대응 | 용도 |
| --- | --- | --- | --- |
| `--paper` | `#ECE4D5` | 한지 아이보리 | 페이지 바탕 |
| `--card` | `#FFFDF9` | 밝은 한지 | 카드·박스 바탕 |
| `--ink` | `#262220` | **먹빛** | 본문 텍스트 (새까만 #000 금지) |
| `--muted` | `#7C7264` | 옅은 먹 | 보조 텍스트·캡션 |
| `--line` | `#E3DAC9` | 한지 결 | 구분선·테두리 |
| `--dancheong` | `#A63B2A` | **단청 적** | 포인트 — 1차 CTA·강조 **한 점만** |
| `--celadon` | `#2F6B63` | **청자 톤** | 보조 포인트 — 링크·배지·아이콘 |
| `--accent-soft` | `#F3E6DC` | 살구빛 한지 | 포인트의 옅은 배경(호버·하이라이트) |

**규칙**

1. **단청 적은 좁게, 한 점만** — 화면당 1차 CTA(합류하기) 등 1~2곳. 넓은 면적 금지 (브랜드 §8: "포인트 컬러는 넓게 칠하지 말고 작게").
2. 채도 높은 형광·원색 **금지**. 그라데이션·네온 효과 금지.
3. **대비 기준**: 본문(`--ink` on `--paper`) 대비율 ≥ 7:1, 보조 텍스트 ≥ 4.5:1 (WCAG AA). 단청 적 위 텍스트는 흰색만.
4. 다크 모드는 만들지 않는다(한지의 밝은 물성이 정체성) — `color-scheme: light` 명시.

## 2. 타이포그래피

| 항목 | 규격 |
| --- | --- |
| 서체 | **Pretendard** (뷰어와 동일 CDN, 비차단 로드) → 폴백: Apple SD Gothic Neo · Noto Sans KR · system-ui |
| 본문 | 17px / 행간 1.78 (모바일 16.5px) — 뷰어에서 검증된 가독 규격 |
| 제목 | h1 1.9rem·800 / h2 1.4rem·700 / h3 1.15rem·700, 자간 −0.02em |
| 인용·약속 문구 | 본문보다 크게(1.15em)·먹빛·좌측 한지 결 테두리 — 제1약속 등 "한 문장이 먼저"(보이스 원칙 4) |
| 숫자 강조 | 0원·30% 등 핵심 수치는 **굵게 + 단청 적 금지**(과장 광고 톤 방지 — 먹빛 굵기로만 강조) |

- **한글이 주인공** — 영문 장식 타이틀 금지, 영문은 경로·고유명에만.
- 줄 길이: 본문 최대 폭 **744px** (뷰어 리더 폭 계승) — 여백으로 숨 쉬게.

## 3. 레이아웃 · 여백

- 기본 리듬: **8px 그리드** (8/16/24/32/48/64).
- 섹션 사이 여백 넉넉하게(데스크톱 96px·모바일 64px — 예시) — "빽빽하게 채우기보다 여백으로 숨 쉬게, 한옥의 비움을 닮게"(브랜드 §8).
- 홈은 [`홈-랜딩-카피`](홈-랜딩-카피.md)의 **블록별 레이아웃 노트**(도식·타임라인·아코디언·sticky CTA)를 그대로 따른다 — 이 가이드는 그 노트의 시각 규격을 정의할 뿐, 구조를 바꾸지 않는다.
- 반응형 브레이크포인트: **860px**(모바일 전환)·**1140px**(와이드) — 뷰어와 동일.

## 4. 컴포넌트 규격

| 컴포넌트 | 규격 |
| --- | --- |
| **1차 CTA 버튼** ("함께하기/합류하기") | 단청 적 배경 + 흰 글자 · radius 10px · 패딩 14×28px · 호버 시 밝기 +6% — **화면당 1개** |
| 2차 버튼 | 투명 배경 + 먹빛 테두리 · 호버 시 `--accent-soft` 배경 |
| **모바일 sticky CTA** | 하단 고정 '합류하기' 바(사이트맵 정본) — 높이 56px, 단청 적, safe-area 대응 |
| 헤더 | 상단 고정 · 한지 바탕 + 하단 1px `--line` · 로고(좌) + 내비(우) · 모바일은 햄버거 |
| 카드 (작가·작품·소식) | `--card` 바탕 · radius 14px · 그림자 은은하게(`0 6px 22px rgba(60,40,20,.10)`) · 사진은 자연광 컷 |
| 아코디언 (FAQ·문의) | 먹빛 질문 + `--line` 구분 · 열림 아이콘은 청자 톤 · `<details>` 기반(JS 없이 동작) |
| 인용/약속 블록 | 한지 결 배경(`#FAF5EC`) + 좌측 4px 청자 테두리 |
| **예시 배지** | `예시` 텍스트 배지 — `--celadon` 테두리·글자, 배경 없음. **미확정 수치(수수료 구간·4주·8주 등) 옆에 필수 부착** — 렌더 시 예시 라벨 보존 원칙([`기술-사양`](기술-사양.md) §0-3)의 시각 규격 |
| TBD/준비 중 표기 | `--muted` 색 + "준비 중" 문구 — `[대괄호]`·빈칸 노출 금지 |
| 푸터 | 먹빛 배경 반전 금지 — 한지 톤 유지 · 법적 링크 3종 + 비제휴 고지 문구 자리 |

## 5. 사진 · 일러스트

- **사진 원칙** (브랜드 §8): 자연광 · 한옥 배경 · 작품이 '상품'이 아니라 '작품'으로. 형광등·흰 배경 누끼 컷 금지.
- 사진 확보 전: 기존 **SVG 브랜드 일러스트**(viewer/assets 26종)와 헐버트 박사 퍼블릭 도메인 사진을 대체재로 사용 — 촬영 필요 컷은 [`시장-벤치마크-개관-전략`](시장-벤치마크-개관-전략.md) 체크리스트의 사진-촬영-리스트로 관리.
- 이미지 비율: 히어로 16:7(뷰어 계승) · 카드 4:3 · `alt` 텍스트 필수.

## 6. 말과 화면의 결 (마이크로카피 연동)

- 버튼·라벨·빈 상태 문구도 [`브랜드-정의서`](브랜드-정의서.md) §6·§9를 따른다 — **'모시다' 금지 · 과장 금지 · 미확정은 예시 배지**.
- 빈 상태(작가들 페이지 등): "첫 동료를 기다립니다" 톤 — 결핍이 아니라 초대로.
- 오류·완료 화면(폼): 응답 기한 명시(예: "3일 안에 연락드립니다") — 정직 원칙의 UI 구현.
