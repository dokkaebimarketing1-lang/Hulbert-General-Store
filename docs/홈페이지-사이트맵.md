# 헐버트잡화점 — 홈페이지 사이트맵 (정보구조 / IA)

> 이 사이트의 **목적은 단 하나 — 우리와 함께할 '작가(동료)'를 모으는 것**입니다. *작가가 극(極) 핵심.*
> ❌ **고객·외국인 관광객은 이 사이트의 대상이 아닙니다.** (매장 고객용·다국어는 필요 시 향후 별도 프로젝트)
> 글은 우리가 이미 만든 문서에서 그대로 가져옵니다.

**버전 v3.0 · 확정 (2026-06)** &nbsp;|&nbsp; ⚠️ 작가 중심으로 재정렬 — **방문/고객·English(외국인) 페이지를 사이트에서 제외**, 홈을 '합류 랜딩'으로.

## 1. 누가 와서, 어떻게 움직이나 (오직 작가)

**🎯 예비 작가 — 유일한 핵심 대상**
> 인스타/소문에서 유입 → **홈(합류 랜딩)** 에서 마음이 움직이고 → **운영 정책·규칙**으로 "정말 믿을 만한가" 확인 → **합류하기**로 지원.
> 👉 **전달 → 이해 → 참여.** 사이트 전체가 이 한 줄을 위해 존재합니다.

**🤝 이미 함께하는 작가 (동료)**
> 작가촌(추후 회원 영역)에서 교육·소식·동료.

> ❌ **고객·외국인 관광객은 타깃이 아닙니다.** 서촌의 풍부한 유동인구는 *'작가에게 좋은 점'(작품이 잘 보이고 팔린다)* 으로만 소개하고, 고객·관광객을 직접 겨냥한 페이지는 만들지 않습니다.

## 5. 공개 vs 비공개

- **🌐 공개**: 위 표의 페이지 전부. 특히 **운영 정책·약관까지 떳떳이 공개**(가치 = 투명).
- **🔒 비공개(내부)**: `사업계획서` · `정관-거버넌스` · `상표-IP-보호-메모` · `성과지표-KPI` · `운영-매뉴얼`. 입점 동의서는 약관만 공개·서명은 합류 시.

## 7. 설계 원칙 (왜 이렇게)

1. **오직 작가에 집중.** 고객·관광객 마케팅은 이 사이트의 목적이 아니다 — 모든 페이지가 '작가 합류'를 향한다.
2. **홈이 곧 합류 설득.** 첫 화면부터 작가에게 "왜 함께해야 하나"를 말한다(클릭 낭비 없음).
3. **규칙을 숨기지 않는다(투명).** 수수료·정산·퇴점을 공개 페이지로 — 신뢰가 곧 합류율.
4. **읽기 쉽게 = 전달 → 이해 → 참여.** 카드·도식·타임라인. 복지수수료는 그림 한 장.
5. **모바일 우선 · 따뜻한 톤 · 실제 작가 사진.**
6. **기존 문서 재활용 · 작게 열고 키운다.**
