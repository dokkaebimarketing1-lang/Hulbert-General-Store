# 헐버트잡화점 — 홈페이지 사이트맵 (정보구조 / IA)

> 공개 웹사이트의 **페이지 구조와 내비게이션**을 정의합니다. 메인은 **작가 모집**, 손님·외국인·브랜드 섹션이 이를 받칩니다.
> 디자인은 별도(준비됨). 각 페이지의 **콘텐츠는 기존 문서에서 그대로 가져옵니다**(재활용) — 출처를 페이지별로 매핑했습니다.

> ⚠️ 정책·사업자 정보 등 일부는 법적 필수이나 아직 **TBD**(아래 푸터 섹션). 지원 폼·SNS 링크는 채널 확정 후 연결.

## 2. 글로벌 내비게이션 (상단 고정)

- **왼쪽**: 로고 — 헐버트잡화점
- **가운데/오른쪽**: 소개 · **작가 모집** · 방문 안내 · 작가촌 · EN
- **CTA 버튼**(강조): **「작가 지원하기」** → `/artists/apply`
- 모바일: 햄버거 메뉴 + CTA 버튼 상시 노출

## 4. 공개하지 않는(내부·비공개) 문서

웹사이트에 올리지 **않는** 백오피스 자료입니다.

- `사업계획서` · `정관-거버넌스` · `상표-IP-보호-메모` · `성과지표-KPI` · `운영-매뉴얼` → **내부 전용**
- `입점-동의서` → 공개 X. **입점 확정 시 작가에게 개별 제공**(서명용)
