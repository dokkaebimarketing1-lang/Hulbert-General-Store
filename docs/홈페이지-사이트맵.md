# 헐버트작가촌 — 사이트맵 (정보구조 / IA)

> **지금 만드는 이 사이트 = 헐버트작가촌.** 우리와 함께할 작가를 *모으고(모집)*, 함께 *자라는(커뮤니티)* **작가 전용 사이트**입니다.
> ⚠️ **헐버트잡화점(매장·고객)과 외국인은 이 사이트의 대상이 아닙니다.** *작가가 극(極) 핵심.*
> 전략·취지는 [`헐버트작가촌-기획.md`](헐버트작가촌-기획.md), 페이지 글은 기존 문서에서 재활용.

**버전 v4.0 · 확정 (2026-06)** &nbsp;|&nbsp; 사이트 정체성을 **'헐버트작가촌'**으로 정립 + **회원(입점 작가) 영역**(아카데미·매거진·사랑방) 통합.

> 이 사이트는 두 겹입니다 — **① 공개(예비 작가 모집)** = 지금의 핵심, **② 회원(입점 작가 커뮤니티)** = 단계적으로 확장.

## 1. 누가 와서, 어떻게 움직이나 (오직 작가)

**🎯 예비 작가 — 가장 핵심 (공개 영역)**
> 인스타/소문 → **홈(합류 랜딩)** 에서 마음이 움직이고 → **운영 정책·규칙**으로 신뢰 확인 → **합류하기**.
> 👉 **전달 → 이해 → 참여.**

**🤝 입점한 작가 — 동료 (회원 영역)**
> 로그인 후 **아카데미(교육) · 매거진(이야기) · 사랑방(커뮤니티)** 에서 배우고, 소통하고, 함께 자란다.

> ❌ 고객·외국인 관광객은 대상이 아닙니다. 서촌 유동인구는 *'작가에게 좋은 점'* 으로만 언급.

## 5. 공개 vs 비공개

- **🌐 공개(누구나)**: 공개 영역 8개 페이지. 운영 정책·약관까지 떳떳이 공개(=투명).
- **👥 회원(로그인)**: 회원 영역 4개 — **입점 작가 전용**.
- **🔒 내부 전용(사이트 X)**: `사업계획서`·`정관-거버넌스`·`상표-IP-보호-메모`·`성과지표-KPI`·`운영-매뉴얼`.

## 7. 설계 원칙

1. **오직 작가에 집중** — 고객·관광객 마케팅은 목적이 아니다.
2. **홈이 곧 합류 설득** — 첫 화면부터 "왜 함께해야 하나".
3. **규칙은 투명 공개** — 신뢰가 곧 합류율.
4. **읽기 쉽게 = 전달 → 이해 → 참여.**
5. **모집(공개) 먼저, 커뮤니티(회원) 나중** — 작게 열고 키운다.
6. **기존 문서 재활용**(13종 연결).
