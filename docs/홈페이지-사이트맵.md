# 헐버트잡화점 — 홈페이지 사이트맵 (정보구조 / IA)

> 이 문서는 우리 홈페이지가 **어떤 페이지로, 어떻게 이어지는지**를 *누구나 한눈에* 이해하도록 그린 '지도'입니다.
> 메인 목적은 단 하나 — **작가(동료)의 합류**. 손님·외국인·브랜드 페이지가 이를 받칩니다.
> 디자인은 준비돼 있고, 각 페이지의 **글은 우리가 이미 만든 문서에서 그대로 가져옵니다**(칸마다 출처를 적어 뒀어요).

**버전 v2.0 · 확정 (2026-06)** &nbsp;|&nbsp; 친절·가독성 강화판 — *방문자 여정 · 설계 원칙 · 작가들 페이지* 추가.

## 1. 누가 와서, 어떻게 움직이나 (방문자 여정)

사이트는 '페이지 묶음'이 아니라 **사람이 지나가는 길**입니다. 네 부류의 손님을 이렇게 안내해요.

**🎨 예비 작가 — 가장 중요한 손님**
> 홈/인스타에서 유입 → **팀 모집(`/join`)** 에서 마음이 움직이고 → **운영 정책·규칙(`/join/policy`)** 으로 "정말 믿을 만한가" 확인 → **합류하기(`/join/apply`)** 로 지원.
>
> 👉 **전달 → 이해 → 참여.** 이 한 줄이 사이트 전체의 뼈대입니다.

**🛍️ 손님 (서촌 방문객·관광객)**
> 홈 → **작가들/작품 둘러보기(`/makers`)** 로 매력을 느끼고 → **방문 안내(`/visit`)** 로 위치·시간 확인 → 방문.

**🌏 외국인 관광객**
> **English(`/en`)** 로 바로 → 방문하거나, 작가라면 합류로.

**🤝 이미 입점한 작가 (동료)**
> **작가촌(`/community`)** 에서 교육·소식·동료를 만남. (차차 회원 전용 영역으로)

## 3. 글로벌 내비게이션 & 공통 요소

- **상단 고정 헤더**: 로고(→홈) · 소개 · **팀 모집** · 작가들 · 방문 안내 · 작가촌 · EN · **[ 합류하기 ]**
- **`/join` 안의 하위 탭**: 랜딩 · **운영 정책·규칙** · 입점 약관 · 합류하기
- **sticky 「합류하기」 버튼** — 스크롤 내내 따라오게(모바일은 하단 고정). *언제든 한 번에 지원.*
- **KO / EN 언어 토글**
- **친절 장치** — 모든 페이지 제목 아래 *'이 페이지는 무엇인가' 한 줄 안내*. 길 잃지 않게.
- **동의 배너** — 합류 폼의 개인정보 수집 관련 쿠키·동의 고지(법적).

## 5. 공개 vs 비공개 (무엇을 사이트에 올리나)

**🌐 공개 페이지** — 위 4장 표 전부. 특히 **규칙을 숨기지 않습니다**: 운영 정책(`/join/policy`)·입점 약관(`/join/terms`)을 떳떳이 공개. *(우리 가치 = 투명)*

**🔒 비공개(내부 전용)** — 사이트에 올리지 않는 백오피스 자료:
- `사업계획서` · `정관-거버넌스` · `상표-IP-보호-메모` · `성과지표-KPI` · `운영-매뉴얼`
- `입점-동의서`는 **약관 본문만 공개 열람**, **서명은 합류 확정 시** 개별 진행.

## 7. 왜 이렇게 짰나 (설계 원칙) — *친절 설명*

1. **메인은 하나, 합류.** 백화점식으로 늘어놓지 않고 모든 길을 「합류하기」로 모읍니다. 헤매지 않게.
2. **규칙을 숨기지 않는다(투명).** 수수료·정산·퇴점 같은 '돈·규칙'을 공개 페이지로. 신뢰가 곧 합류율입니다.
3. **읽기 쉽게 = 전달 → 이해 → 참여.** 긴 글 대신 카드·도식·타임라인. 복지수수료는 '그림 한 장'으로.
4. **모바일 우선 · 따뜻한 톤 · 실제 작가 사진.** 채용 페이지 정석(토스·Airbnb 벤치마크)을 따릅니다.
5. **이미 만든 문서를 재활용.** 새로 쓰지 않고 13개 문서를 페이지에 '연결'만 — 일관성↑, 작업량↓.
6. **작게 열고 키운다.** 1차 4페이지로 빠르게 열고, 작가촌·소식·EN·정책 페이지로 확장.
