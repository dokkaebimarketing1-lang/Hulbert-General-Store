# 헐버트작가촌 — 사이트맵 (정보구조 / IA)

> 이 사이트 = **헐버트작가촌** (작가 전용 사이트). **지금 만드는 범위 = '예비 작가 모집' 단계** 하나에 집중합니다.
> ⚠️ 입점 작가 **회원·커뮤니티 영역(아카데미·매거진·사랑방·내 정산)은 '다음 시점'** 으로 분리 — 전략은 [`헐버트작가촌-기획.md`](헐버트작가촌-기획.md).
> ⚠️ **헐버트잡화점(매장·고객)·외국인은 이 사이트의 대상이 아닙니다.** *작가가 극(極) 핵심.*

**버전 v5.0 · 확정 (2026-06)** &nbsp;|&nbsp; 회원·커뮤니티 영역을 **'다음 시점'으로 분리**, 현재 사이트맵은 **모집 단계**에 집중.

## 0. 한눈에 보기 (모집 사이트 한 장 요약)

```
                    [ 헐버트작가촌 · 작가 모집 ]
                          홈 = 합류 랜딩 ★
   ┌────────┬───────────┬─────┼──────┬────────┬────────┐
  소개    운영정책·규칙   약관   작가들    소식      문의·FAQ

                         │
                    합류하기 ★  /apply
  상단 어디서나: [ 합류하기 ] · 로고→홈
  푸터: 찾아오시는 길(서촌) · 개인정보처리방침* · 이용약관* · 사업자정보* · SNS   (* 법적 필수)
```
★ = 모든 길이 **「합류하기」**로. **홈 자체가 합류 설득 페이지.**

## 1. 누가 와서, 어떻게 움직이나 (오직 예비 작가)

**🎯 예비 작가 — 유일한 핵심 대상**
> 인스타/소문에서 유입 → **홈(합류 랜딩)** 에서 마음이 움직이고 → **운영 정책·규칙**으로 "정말 믿을 만한가" 확인 → **합류하기**로 지원.
> 👉 **전달 → 이해 → 참여.** 이 사이트는 오직 이 한 줄을 위해 존재합니다.

> ❌ 고객·외국인 관광객은 대상이 아닙니다. 서촌의 풍부한 유동인구는 *'작가에게 좋은 점'(작품이 잘 보이고 팔린다)* 으로만 소개.

## 5. 공개 vs 비공개

- **🌐 공개**: 위 8개 페이지 전부. 운영 정책·약관까지 떳떳이 공개(가치 = 투명).
- **🔒 비공개(내부, 사이트 X)**: `사업계획서` · `정관-거버넌스` · `상표-IP-보호-메모` · `성과지표-KPI` · `운영-매뉴얼`. 입점 동의서는 약관만 공개·서명은 합류 시.

## 7. 설계 원칙

1. **오직 작가 모집에 집중** — 고객·관광객·커뮤니티는 지금 범위가 아니다.
2. **홈이 곧 합류 설득** — 첫 화면부터 "왜 함께해야 하나".
3. **규칙을 숨기지 않는다(투명)** — 신뢰가 곧 합류율.
4. **읽기 쉽게 = 전달 → 이해 → 참여.** 카드·도식·타임라인.
5. **모바일 우선 · 따뜻한 톤 · 실제 작가 사진.**
6. **기존 문서 재활용 · 작게 열고 키운다.**
